Location API

The Location API takes two Postcode inputs from the user, a "To" postcode and a "From" postcode, and returns to the user the location information of the two postcodes and sends the longitude and latitude information to the Transport API.

Transport API

The Transport API takes the latitude and longitude values from the Location API and return the times for all valid methods of transport between the two points. It then finds the shortest distance to the nearest station for both start and end points.
